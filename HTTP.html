-Http

The Hypertext Transfer Protocol(HTTP) is the abbreviation of HyperText Transfer Protocol. It is the protocol that is applicable in TCP/IP A protocol that includes a group of foundation protocols for the internet. The development of HTTP was initiated by Tim Berners-Lee at CERN in 1989. Here is an example of HTTP: Example: A client (browser) sends an HTTP request to the server; then the server returns a response to the client. The response contains status information about the request and may also contain the requested content.

-Https

HTTPS is the abbreviation of HyperText Transfer Protocol Secure. It is the combination between HTTP protocol and secure protocol SSL or TLS which allows the information exchange on the Internet to happen in a secure way. HTTPS protocol is usually used in sensitive deals which need high security. Most of the websites we used are using HTTPS protocol, for example, Google and Facebook. We need to use "HTTPS://" before entering those website pages which means the data we entered in those websites are blocked to any third party.
